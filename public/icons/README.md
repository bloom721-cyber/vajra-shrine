# icons/

Interface SVG icons only. Sacred-symbol artwork belongs in `/images/symbols/`.

- Format: SVG, 24×24 viewBox, 1.5 px stroke, `currentColor` (tinted antique gold in CSS)
- Optically centered, no embedded fills of fixed color, no raster data
- Subfolders: navigation, offerings, settings, accessibility, gestures, status,
  notifications, system, menu, actions
- Name pattern: `<group>-<meaning>.svg` → `nav-shrine.svg`, `offering-water.svg`
